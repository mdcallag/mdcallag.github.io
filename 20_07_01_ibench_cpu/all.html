Introduction

This is a report for the insert benchmark with 40M docs and 8 clients. It is generated by scripts. An overview of the insert benchmark is here.

The test server is a c3.8xlarge host with 20,000 EBS PIOPs, 16 CPU cores (HT disabled). Clients and the DBMS shared the same host. The DBMS configurations are here.

The tested DBMS are:

  • mo4019.c5b40 - MongoDB 4.0.19 with the c5b40 config
  • mo428.c5b40 - MongoDB 4.2.8 with the c5b40 config
  • mo440rc13.c5b40 - MongoDB 4.4.0 rc13 with the c5b40 config
  • pg123.c8b40 - Postgres 12.3 with the c8b40 config
  • my8020.c10b40 - MySQL 8.0.20 with InnoDB and the c10b40 config
  • rx56.c7b40 - MySQL 5.6.35 with MyRocks and the c7b40 config
